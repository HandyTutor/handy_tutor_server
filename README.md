# Handy Tutor Server
## How to install
<pre>
  $ git clone https://github.com/HandyTutor/handy_tutor_server
  $ git cd handy_tutor_server
  $ pip install virtualenv
  $ virtualenv env
  $ source env/bin/activate
  $ pip install flask
  $ pip install pymysql
</pre>

## How to run
<pre>
  $ cd ~/handy_tutor_server
  $ source env/bin/activate
  $ python server.py
</pre>
