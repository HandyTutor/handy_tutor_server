# Handy Tutor Server
## How to install
<code>
  git clone https://github.com/HandyTutor/handy_tutor_server
  </code>
<code>
  git cd handy_tutor_server
  </code>
<code>
  pip install virtualenv
  </code>
<code>
  virtualenv env
  </code>
<code>
  source env/bin/activate
  </code>
<code>
  pip install flask
  </code>
<code>
  pip install pymysql
  </code>
## How to run
<code>
  cd ~/handy_tutor_server
  </code>
<code>
  source env/bin/activate
  </code>
<code>
  python server.py
  </code>
